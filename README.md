# 久坐提醒 (Sedentary Reminder)

一款帮助用户管理工作/休息时间的桌面提醒工具。

## 功能特性

- **工作计时器** - 设置工作时间，到时提醒休息
- **休息提醒** - 全屏 overlay 提醒，强制用户离开电脑休息
- **多屏支持** - 休息提醒会在所有显示器上同步显示
- **键盘鼠标锁定** - 休息期间可锁定键盘鼠标，强制休息（需管理员权限）
- **系统托盘** - 后台运行，通过托盘菜单控制
- **自动提权** - 启动时自动请求管理员权限，确保锁定功能正常工作

## 使用方法

### 首次运行

程序启动时会自动请求管理员权限（用于锁定键盘鼠标功能）。

### 配置提醒

1. 设置**工作时间**（分钟）：默认 60 分钟
2. 设置**休息时间**（分钟）：默认 5 分钟
3. 可选择是否启用**键盘鼠标锁定**

### 开始使用

点击「开始」按钮后：
- 主窗口会自动隐藏
- 右下角出现工作计时器窗口（始终置顶）
- 计时结束后，所有显示器显示全屏休息提醒
- 休息结束后自动开始新的工作周期

### 托盘菜单

右键点击系统托盘图标可：
- 显示主窗口
- 隐藏主窗口
- 退出程序

## 系统要求

- Windows 10/11
- WebView2 运行时（首次运行时会自动下载安装）

## 下载安装

预构建的安装包可在 [GitHub Releases](https://github.com/bill756/Reminder-Tauri/releases) 页面下载。

## 灵感来源

本项目受 [Sedentary-reminder](https://github.com/wjbgis/Sedentary-reminder) 启发开发。

## 技术栈

- Tauri 2.x (Rust 后端)
- React + TypeScript (前端)
- Vite (构建工具)

## 构建

```bash
# 安装依赖
npm install

# 开发模式
npm run tauri dev

# 生产构建
npm run tauri build
```
